Fancypants Sliding Modal Demo

This is page demonstrates how a modal dialog can be slid in from above the viewable page area, while at the same time having a transparent overlay fade in. Closing the modal dialog (by pressing escape or clicking the close button on the dialog itself) will cause the modal to slide back up to the hidden position, while at the same time fading the overlay back to hidden. The modal itself is an Angular JS directive, and I'm using LESS CSS for some of the more complicated CSS rules. I highly recommend you check them out.
